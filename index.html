音频播放操作

发表于 2016-03-21

iOS中
框架：播放音频需要用到AVFoundation.framework框架
首先先导入依赖库 说明：AVFoundation.framework框架中的东西转换为CF需要使用桥接。

1.获得音效文件的路径

　　NSURL *url = [[NSBundle mainBundle] URLForResource:@”声音.wav”（写你文件名） withExtension:nil];

2.加载音效文件，得到对应的音效ID

　　SystemSoundID soundID = 0;

　　AudioServicesCreateSystemSoundID((__bridge CFURLRef)(url), &soundID);

3.播放音效

　　AudioServicesPlaySystemSound(soundID);

就这么简单的三步走 音频加入小动画 效果更好

CocoaPods 导入第三方库找不到 头文件

发表于 2016-03-07

最近一直在了解MVVM架构模式，也知道了ReactiveCocoa框架对MVVM实现的便利与优雅，但是CocoaPods导入ReactiveCocoa框架后，却出现一个问题，就是引入头文件的时候说找不到头文件
解决方法如下：

1，找到TARGETS -> Build Settings -> SearchPaths -> User Header Search Paths 在后面的空白处双击
2， 点击 ‘+’号，添加一个新的键为${SRCROOT}，值设置为recursive
现在再来看一下导入的头文件,问题就解决了 so easy
